# Identity and AI Agents

## Overview
AI agents act as non-human identities and must be governed accordingly.

---

## Key Concepts

### AI Agents as Identities
- Represent users or systems
- Execute tasks autonomously

### Service Principals
- Used for authentication
- Scoped permissions

### Least Privilege
- Limit access to required resources only

---

## Access Control

- Role-based permissions
- API access restrictions
- Token management

---

## Monitoring

- Track agent activity
- Audit actions performed

---

## Key Principle

**AI agents must be treated as identities with strict access controls and accountability.**
